Workshop (Training)

This is a FedSIS used for workshops and training. It is used when a new clean empty system is desired. This training FedSIS is usually destroyed and re-created every once in a while.

flash_on

Sandbox (Testing)

This is a FedSIS used for testing, practice, training, etc. This FedSIS is generally sync with the production so you can try things out when not sure before attempting it in Production system.

group

Production (Real)

This is the FedSIS used in production and it the final correct data. You should take a bit more care with this one as it affects the data you will eventually be submitting and using as your real data.



Product Details

The FedSIS tool is a component of the large FedEMIS project. It is the component meant to support the schools in their day-to-day operations and data management.
